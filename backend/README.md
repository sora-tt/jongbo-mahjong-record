To develop locally (no Vercel account required):

```
npm install
npm run dev
```

```
open http://localhost:3000
```

To use a custom port:

```
PORT=8080 npm run dev
```

Type check:

```
npm run typecheck
```

If you want to use Vercel CLI workflows:

```
npm install
vc dev
vc build
vc deploy
```
